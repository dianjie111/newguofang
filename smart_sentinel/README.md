# 边境 / 重要设施智能哨兵模拟系统

## 项目简介

本系统是一个面向边防前哨和重要设施的轻量化智能警戒模拟软件，通过自动目标检测与威胁评估，辅助哨兵完成24小时不间断警戒任务。

## 核心功能

1. **目标检测与分类**：识别图片中的人、车、动物等目标
2. **威胁等级评估**：根据目标类型、数量、时间等因素判定威胁（高/中/低/无）
3. **处置建议输出**：针对不同等级给出具体行动建议
4. **日志记录与保存**：保存每次分析的时间、图片、结果，便于追溯

## 技术特点

- **轻量离线**：无需网络、无需高性能硬件
- **自主可控**：Python实现，可适配国产系统
- **扩展性强**：未来可接入真实摄像头、升级检测算法
- **直观易用**：图形化界面，操作简单

## 系统要求

- **开发语言**：Python 3.9+
- **依赖包**：
  - OpenCV (`pip install opencv-python`)
  - PIL (`pip install Pillow`)
- **运行环境**：Windows / Linux / 国产操作系统
- **硬件要求**：普通办公电脑即可

## 安装与运行

### 1. 安装依赖

```bash
pip install opencv-python Pillow
```

### 2. 下载模型文件

请参考 [models/README.md](models/README.md) 中的说明，下载并放置模型文件到 `models` 目录。

### 3. 运行系统

```bash
cd smart_sentinel
python main.py
```

## 使用方法

1. **上传图像**：点击"上传图像"按钮，选择要分析的图像文件
2. **输入位置**：在"位置"输入框中填写图像拍摄的位置信息（可选）
3. **分析图像**：点击"分析图像"按钮，系统会自动检测目标并评估威胁等级
4. **查看结果**：系统会显示检测结果、威胁等级和处置建议
5. **查看日志**：点击"查看日志"按钮，查看历史分析记录

## 应用场景

- **边境线哨所**：哨兵使用望远镜或侦察设备观察边境方向
- **弹药库周界警戒**：周界摄像头定时采集画面
- **雷达站入口检查**：哨兵对进入人员/车辆拍照
